How to select headlines and paragraphs in a given section: to complete these exercise you will need to apply seperate font-family rules to headings and paragraphs within a given section. This means you need to select each separately. In order to accomplish this, you need to learn another CSS selector.

The following syntax selects all p tags inside the introduction section:
#introduction p { }
More generally, any selector followed by a space followed by another selector will select any elements defined by the second selector nested inside any elements described by the first selector. For more about selectors, refer to this. For the examples below, to select the heading for a given section id use:
#sectionID h2
For the paragraphs:
#sectionID p

Golisano Children's Hospital Pediatric Practice

We are the area’s largest pediatric clinic, serving over 12,000 children in the greater Rochester area. Our practice offers exceptional healthcare services for children from birth to age 21. We also treat a variety of complicated medical conditions. We have doctors who are experts in the care of asthma, ADHD, obesity, and lead poisoning.

President Barack Obama

Barack H. Obama is the 44th President of the United States.
